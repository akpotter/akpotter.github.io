Worldwide Distribution of the Backdoor-affected Netcore/Netis(磊科) Router2014-8-8

Recently we found a remote access backdoor of Netcore/Netis devices, which can bypass the authorization and get full control.

According to the firmware security test, affected models included the latest one(Almost all of the router products from Netcore/Netis).

The worldwide map below is based on Zmap UDP port probe module. There might be exist about 2000,000 backdoor embeded devices worldwide.
